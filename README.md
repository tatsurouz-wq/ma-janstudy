# 雀学（じゃんがく）

麻雀のルールと点数計算を、牌を動かしながら学べるウェブアプリです。社内麻雀大会に向けて、未経験者がリーチ麻雀の基礎から点数計算までを身につけることを目的としています。

## 機能

| モード | 内容 |
|---|---|
| レッスン | 全8章のインタラクティブ教材。牌の仕分け・手牌の分解・待ち当てなどを実際に牌を動かして学ぶ |
| クイズ | 待ち当て・役当て・点数当ての3カテゴリ。間違えた問題は次回優先的に再出題 |
| 点数計算機 | 牌を選んで手牌を作ると役・符・点数を自動判定し、計算過程をステップ解説。副露・槓・ドラ・本場に対応 |
| ミニ実戦 | 一人打ちの練習対局。ツモ→打牌→リーチ→ツモ和了の流れを体験。向聴数・受け入れ表示などの学習補助つき |
| 3D実戦体験 | 全自動卓（AMOS風）の半荘ダイジェストを1人称シネマティック3Dで体験。自動洗牌・山のせり上がり・配牌・リーチ・和了・精算・終局順位までを学習字幕つきで観られる。一時停止・倍速・チャプタージャンプ対応（WebGL2必須） |

## 起動方法

Node.js 20以上が必要です。

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:5173` を開いてください。

ビルドして配信する場合:

```bash
npm run build
npm run preview
```

## 開発コマンド

```bash
npm run test           # ユニットテスト（Vitest）
npm run test:coverage  # カバレッジ計測（core 90% / 全体 80% の閾値つき）
npm run lint           # ESLint（mutation禁止・console禁止）
npm run typecheck      # TypeScript型チェック
npm run e2e            # E2Eテスト（Playwright。初回は npx playwright install chromium）
```

## 採用ルール（デフォルト）

一般的なアリアリルール準拠: 喰いタンあり / 後付けあり / 赤ドラ各1枚 / 一発・裏ドラあり / 切り上げ満貫なし / 平和ツモ20符 / 七対子25符 / 連風牌の雀頭2符 / ダブル役満なし / 数え役満あり。

## アーキテクチャ

- `src/core/` — Reactに依存しない純関数ロジック層（牌・手牌分解・向聴数・役判定・符計算・点数算出・ゲーム進行）
- `src/content/` — レッスン教材とクイズ問題の宣言的データ。クイズの正答は実行時にエンジンで算出
- `src/components/` `src/features/` — UI層（牌SVGは自作、画像アセット非依存）
- `src/state/` — Zustandストア（進捗はlocalStorageに保存）
